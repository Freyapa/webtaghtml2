HTML5 Cheat Sheet (ภาษาไทย)

สรุปแท็ก HTML5 สำคัญเกือบทั้งหมด พร้อมความหมาย ตัวอย่างการใช้งาน และแอตทริบิวต์ที่ควรรู้

สารบัญ

เมทาดาทา & โครงสร้างเอกสาร โครงสร้างเนื้อหา (Sectioning) ข้อความ & ลิสต์ อินไลน์เซมานติก การแก้ไขข้อความ สื่อและการฝัง ตารางข้อมูล ฟอร์ม อินเทอร์แอกทีฟ & เว็บแอป สคริปต์ แท็กที่เลิกใช้/ควรหลีกเลี่ยง แอตทริบิวต์สากล (Global attributes)

เมทาดาทา & โครงสร้างเอกสาร

แท็กพื้นฐานสำหรับตั้งค่าหน้าเว็บและข้อมูลเบื้องหลัง

แท็กความหมายชนิดแอตทริบิวต์ที่พบบ่อย
<html>เอกสาร HTML ทั้งหมด เป็นรากของหน้าเว็บBlocklang, dir
<head>ส่วนข้อมูลเบื้องหลังของเอกสาร เช่น meta, title, link, scriptBlock
<body>ส่วนเนื้อหาที่ผู้ใช้เห็นจริงBlock
<base>กำหนดฐาน URL และ target เริ่มต้นของลิงก์ทั้งหมดInlinehref, target
<link>เชื่อมโยงไฟล์ภายนอก เช่น CSS, icon, preloadInlinerel, href, as
<meta>ข้อมูลเมตา เช่น charset, viewport, descriptionInlinecharset, name, content
<title>ชื่อหน้าเว็บที่แสดงบนแท็บเบราว์เซอร์Inline
ตัวอย่าง 
<!DOCTYPE html>
<html lang="th">
  <head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>ตัวอย่าง</title>
    <link rel="stylesheet" href="styles.css">
  </head>
  <body>...</body>
</html>

โครงสร้างเนื้อหา (Sectioning)

แท็กความหมายชนิดแอตทริบิวต์
<body>ส่วนเนื้อหาที่ผู้ใช้เห็นจริงBlock
<header>ส่วนหัวของหน้า/บทความ มักมีโลโก้ เมนู ชื่อเรื่องBlock
<nav>ส่วนของลิงก์นำทางหลักBlock
<main>เนื้อหาหลักของหน้า (มีได้เพียงหนึ่งเดียว)Block
<section>กลุ่มเนื้อหาที่มีหัวเรื่องของตัวเองBlock
<article>บทความที่เป็นอิสระ สามารถอ่านเดี่ยว ๆ ได้Block
<aside>ข้อมูลเสริม เช่น แถบด้านข้าง โฆษณา กล่องข้อมูลBlock
<footer>ส่วนท้ายของหน้า/บทความ เช่น ลิขสิทธิ์ ลิงก์เพิ่มเติมBlock
<h1–h6>หัวข้อระดับ 1 ถึง 6 (ใช้ h1 เพียงครั้งเดียวต่อหน้า)Blockid
<address>ข้อมูลติดต่อผู้เขียน/องค์กรBlock

แท็กที่เลิกใช้/ควรหลีกเลี่ยง

หลีกเลี่ยงการใช้แท็กเหล่านี้และใช้ CSS/วิธีสมัยใหม่แทน

แท็กสถานะคำแนะนำ
<center>เลิกใช้ใช้ CSS: text-align:center
<font>เลิกใช้ใช้ CSS: font-family / color / size
<marquee>เลิกใช้ใช้ CSS Animations

แอตทริบิวต์สากล (Global attributes)

ใช้ได้กับแท็กส่วนใหญ่:

  • id: ตัวระบุไม่ซ้ำในหน้า
  • class: จัดกลุ่มเพื่อสไตล์/สคริปต์
  • style: ใส่ CSS ตรง ๆ
  • title: ข้อความเมื่อเอาเมาส์ชี้
  • lang, dir: ภาษาและทิศทางข้อความ
  • hidden: ซ่อนองค์ประกอบ
  • tabindex: ลำดับการโฟกัสด้วยแป้นพิมพ์
  • draggable, contenteditable, spellcheck, translate
  • data-*: เก็บข้อมูลกำหนดเอง
  • เหตุการณ์ (Events): onclick, oninput, ฯลฯ
  • ARIA: role, aria-* เพื่อการเข้าถึง
